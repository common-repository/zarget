== Installation ==
1. Extract the “.zip” file and drop the contents in “wp-content/plugins/“ directory of your WordPress installation..
2. Activate Zarget plug-in from the plug-ins page.
3. Login to Zarget and click on “Setup” -> “Integrations”
4. Click on “Generate API key”. Copy the key.
5. Goto the “Zarget” tab in your Wordpress site, paste the key and click on "Connect" button.
